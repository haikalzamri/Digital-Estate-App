# Digital-Estate-App

Digital Estate work programme tracking prototype for capture, approval, dashboard monitoring, and GIS map visualisation.

## Contents

- Mobile-friendly field capture form
- Records review and approval workflow
- Dashboard with planned vs approved actual completion
- Leaflet/OpenStreetMap GIS map with KMZ-derived field polygons
- Exported GeoJSON field layer for GIS tools

## App

Open `work-program-tracker-app/index.html` in a browser, or serve the folder with a local web server for the GIS map view.
